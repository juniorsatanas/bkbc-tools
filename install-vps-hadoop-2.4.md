## install hadoop

```
$ git clone https://github.com/apache/hadoop-common
$ cd hadoop-common
$ git tag -l '*2.4.*'
$ git checkout release-2.4.0
$ cd hadoop-dist
$ mvn package -Pdist,native -DskipTests -Dtar
$ file hadoop-dist/target/hadoop-2.1.0-beta/lib/native/*
```
